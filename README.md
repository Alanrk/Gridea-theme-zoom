# Gridea-theme-zoom
一款简介的Gridea博客主题，示例网站-https://kveln.cn/
## 安装方法
- 克隆项目到本地，或者从百度网站下载
- 复制「 space 」目录到「 Gridea 」主题文件中
- 重启「 Gridea 」客户端
- 在主题中选择「 space 」主题，点击保存
- 预览查看效果
